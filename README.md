# EU5-HRE-Reichskrone-Bugfix

Makes the HRE ruler wear the Reichskrone. Fixes the base game bug where the Emperor defaults to random generic crowns.

## Installation

1. Download the `HRE` folder.
2. Drop it directly into your local mod directory(create mod folder if none exists):
   `Documents\Paradox Interactive\Europa Universalis V\mod\`
3. Enable **HRE Reichskrone** in the in-game Mods menu, click **Apply**.
